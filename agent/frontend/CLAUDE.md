# agent/frontend

Wails v2 + Svelte 5 frontend for the rick-agent desktop app — chat UI, workflow dashboard, and event stream backed by the Go side in `../app.go`.

## Layout
- `package.json` — npm scripts (`dev`, `build`, `preview`) and deps: Svelte 5, Tailwind 4, Vite 6, marked, highlight.js, mermaid
- `vite.config.ts` — Vite + `@sveltejs/vite-plugin-svelte` + `@tailwindcss/vite` plugin; outputs to `dist/`
- `svelte.config.js` — Svelte preprocess via `vitePreprocess()` only (no extra config)
- `tsconfig.json` — strict TS, ESNext, bundler resolution; includes `src/**/*.{ts,d.ts,svelte}`
- `index.html` — Vite entry; loads `/src/main.ts` and Inter from Google Fonts; mounts on `#app`
- `src/` — Svelte source. Subdirs each have their own CLAUDE.md:
  - `src/lib/CLAUDE.md` — components and shared UI
  - `src/stores/CLAUDE.md` — Svelte stores
  - `src/utils/CLAUDE.md` — utility helpers
- `wailsjs/` — auto-generated Wails bindings (top-level copy, DO NOT EDIT)
- `dist/` — Vite build output (gitignored)
- `node_modules/` — gitignored

## Build
- Production: built indirectly via Wails — `cd agent && wails build` (do not run `vite build` directly for releases; Wails wires the embed)
- Dev: `cd agent && wails dev` (hot reload through Wails)
- The `.deb` package is produced from `agent/build/bin/rick-agent` — see root CLAUDE.md "Agent UI" section

## Two wailsjs dirs (gotcha)
- Both `agent/frontend/wailsjs/` AND `agent/frontend/src/wailsjs/` exist and are checked into git
- Both are auto-generated by Wails (`go/main/App.{js,d.ts}`, `go/models.ts`, `runtime/`)
- DO NOT edit either by hand; regenerate via `wails generate module` or `wails build`
- Imports in Svelte files typically come from the `src/wailsjs/` copy (relative paths inside `src/`)
